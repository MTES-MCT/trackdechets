Exporter un bordereau en pdf

Il est possible à tout moment d'obtenir une version pdf d'un BSD (à l'exception d'un BSD à l'état brouillon). L'obtention du pdf se fait en deux temps. Il faut d'abord récupérer un lien de téléchargement grâce à une mutation graphQL en passant en argument l'identifiant du BSD, puis utiliser ce lien pour télécharger le fichier

  • Déchets dangereux
  • DASRI
  • VHU
  • Amiante
  • Fluides Frigo
query {  formPdf(id: "BSD_ID") {    downloadLink  }}
{  "data": {    "formPdf": {      "downloadLink": "https://api.trackdechets.beta.gouv.fr/download?token=form_pdf-xxxxxxxxx-xxxx"    }  }}
query {  bsdasriPd(id: "BSD_ID") {    downloadLink  }}
{  "data": {    "bsdasriPdf": {      "downloadLink": "https://api.trackdechets.beta.gouv.fr/download?token=form_pdf-xxxxxxxxx-xxxx"    }  }}
query {  bsvhuPdf(id: "BSD_ID") {    downloadLink  }}
{  "data": {    "bsvhuPdf": {      "downloadLink": "https://api.trackdechets.beta.gouv.fr/download?token=form_pdf-xxxxxxxxx-xxxx"    }  }}
query {  bsdaPdf(id: "BSD_ID") {    downloadLink  }}
{  "data": {    "bsdaPdf": {      "downloadLink": "https://api.trackdechets.beta.gouv.fr/download?token=form_pdf-xxxxxxxxx-xxxx"    }  }}
query {  bsffPdf(id: "BSD_ID") {    downloadLink  }}
{  "data": {    "bsffPdf": {      "downloadLink": "https://api.trackdechets.beta.gouv.fr/download?token=form_pdf-xxxxxxxxx-xxxx"    }  }}
warning

L'URL a une durée de validité de 10 secondes.
