En plus de l'environnement de production, nous mettons à votre disposition un environnement _sandbox_ permettant d'effectuer des tests. L'environnement _sandbox_ est un clone de l'environnement de production avec une base de données séparée.

:::tip
L'environnement sandbox se distingue par la présence du mot _sandbox_ dans les URLs d'accès et par un bandeau d'avertissement dans l'interface web. Gardez bien en tête l'environnement sur lequel vous êtes lorsque vous effectuez des actions.
:::
