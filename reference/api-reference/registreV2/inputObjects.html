IncomingTexsLineInput​

Contenu d'une ligne pour l'import de terres excavées et sédiments, dangereux et non dangereux entrants

Arguments

NameDescription

reason
RegistryLineReason

Motif

Indique si la ligne doit être modifiée ou supprimée. A laisser vide si c'est une création.

Si aucune valeur n'est renseignée et que la ligne n'a jamais été déclarée (vérification via l'identifiant unique), alors celle-ci est ajoutée comme nouvelle déclaration.

Si aucune valeur n'est renseignée et que la ligne a déjà été déclarée (vérification via l'identifiant unique), alors la ligne sera ignorée.

Si la valeur CANCEL a été renseignée et que la ligne a déjà été déclarée (vérification via l'identifiant unique), alors celle-ci sera supprimée.

Si la valeur EDIT a été renseignée et que la ligne a déjà été déclarée (vérification via l'identifiant unique), alors celle-ci sera modifiée par les nouvelles valeurs.

Si la valeur CANCEL ou EDIT a été renseignée et que la ligne n'a jamais été déclarée (vérification via l'identifiant unique), alors une erreur remontera.

Valeurs :

  • CANCEL
  • EDIT

publicId
ID!

Identifiant unique

Identifiant unique donné à chaque déclaration pour un établissement donné.

Si un import comporte plusieurs mêmes identifiants uniques sans motif, seule la première ligne sera déclarée, les suivantes seront ignorées.

Valeurs :

  • Format libre
  • De 1 à 36 caractères
  • Caractères autorisés : alphanumériques, point (.), tiret (-), slash (/), underscore (_)
