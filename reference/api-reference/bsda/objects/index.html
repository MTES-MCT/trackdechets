InitialBsda​

Fields

NameDescription
id
ID!

Identifiant unique assigné par Trackdéchets. Il est à utiliser pour les échanges avec l'API.

emitter
BsdaEmitter

Émetteur du déchet, qui n'est pas nécessairement le producteur.

destination
BsdaDestination

Destination du déchet, qui peut le réceptionner pour traitement, groupement ou réexpedition. Un nouveau bordereau doit être créé si le déchet connaît un nouveau parcours, par exemple s'il quitte une installation de collecte pour un centre de traitement.

Pour plus de détails sur les différents types de bordereau, voir l'enum BsdaType.

waste
BsdaWaste

Détails du déchet.

weight
BsdaWeight

Quantité totale du déchet en tonnes, il peut s'agir d'une estimation.

packagings
[BsdaPackaging!]!

Liste des contenants utilisés pour le transport du déchet.
