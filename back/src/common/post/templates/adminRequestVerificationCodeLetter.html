Procédure de validation

Pour finaliser l’accès administrateur, vous devez :

  1. Vous connecter à votre compte Trackdéchets;
  2. Accéder à la rubrique Mes établissements;
  3. Cliquer sur Gestion avancée;
  4. Cliquer sur Saisir un code reçu par courrier;
  5. Saisir le siret de l'entreprise et le code mentionné ci-dessus.

Seul vous êtes habilité à saisir ce code. Dès validation, et si votre demande n'a pas été refusée par un administrateur entre-temps, vos droits d’administrateur seront activés.

Si vous estimez que cette demande n’est pas légitime, aucun retour n’est nécessaire de votre part. Pour toute question, vous pouvez contacter l'assistance Trackdéchets.
